# DI01 + DI02 – Media Downloader & Library (Swing, NetBeans Designer)

Run: Right‑click project → **Run** (uses exec-maven-plugin).  
Main class: `gelabert.app.App`.

**DI01_1**: MainFrame (null layout, fixed), menu, Preferences as JPanel, About as modal JDialog, downloader stub.  
**DI01_2**: LibraryPanel with JList (folders), JComboBox (MediaType), JTable (AbstractTableModel) + filters + delete.
